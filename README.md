# Assignment 1 - HTML Basics

This assignment demonstrates the use of basic HTML elements, including hyperlinks, images, and page navigation.

### Features

* Link to Google that opens in a new tab.
* Link to Facebook that opens in the same tab.
* Display of a local image (`abc.png`).
* Display of an image from an online source.
* Link to another page containing a list (`a1pa.htm`).

### Technologies Used

* HTML5

### Learning Outcomes

* Creating hyperlinks
* Opening links in the same and new tabs
* Adding local and online images
* Basic webpage navigation
* Creating and linking list pages
